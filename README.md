# TODO LIST API

## Description

Simple api to a todo list

### Environment

- Database -> MongoDB (mongoose)
- Api -> express app
- Middleweares -> helmet, cors, body-parser, morgan

### Entities 

Every record should have 

- [x] Created At: Datetime
- [x] Updated At: Datetime


- [] user
    - [] _id
    - [] username


- [] todo
    - [] _id
    - [] userId  
    - [] title : string
    - [] description : string
    - [] completed : boolean
